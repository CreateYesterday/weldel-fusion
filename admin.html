GitHub Auto-Deploy
Connect GitHub so every save pushes live to Netlify automatically — no drag and drop ever.

✅ GitHub is connected. Your site deploys automatically when you click Push Live.

📋 One-Time Setup (10 minutes)
1
Create a GitHub account
Go to github.com → Sign up free with your email.
2
Create a new repository
Click New repository → name it weldel-fusion → set to Public → click Create.
3
Upload your site files to GitHub
In your new repo → click uploading an existing file → drag all your files (index.html, admin.html, logo.png, rig.jpg, all .jpg photos, netlify.toml) → click Commit.
4
Connect Netlify to GitHub
In Netlify → Add new site → Import from Git → GitHub → pick your weldel-fusion repo → Deploy. Your domain connects automatically.
5
Create a GitHub Personal Access Token
GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new.
Set: Repository access → weldel-fusion only. Permissions → Contents → Read and Write. Copy the token — you only see it once.
6
Enter your details below and Save
Paste your GitHub username, repo name, and token below. From now on, click Push Live to deploy instantly.
🔑 GitHub Credentials

Your token is stored only in your browser (localStorage). It never leaves your device except when making a deploy commit to GitHub. Use a Fine-grained token scoped only to this repo for maximum security.
