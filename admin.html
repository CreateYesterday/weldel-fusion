GitHub Auto-Deploy
Connect GitHub so Push Live and photo uploads work instantly.
📋 Setup Steps
1
Create a GitHub account
Go to github.com and sign up free.
2
Create repo weldel-fusion
New repository → Public → Create.
3
Upload all site files to GitHub
Upload index.html, admin.html, netlify.toml, and all .jpg/.png photos.
4
Connect Netlify to GitHub
Netlify → Add new site → Import from Git → pick your repo.
5
Create a Personal Access Token
GitHub → Settings → Developer settings → Fine-grained tokens → Generate new. Give Contents Read+Write on your repo only. Copy the token.
6
Paste your details below and Save
Push Live and photo uploads will work from then on.
